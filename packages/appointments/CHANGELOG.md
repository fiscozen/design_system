# @fiscozen/appointments

## 1.1.1

### Minor Changes

- **LIB-2360: Nuovo componente FzAppointments.** Componente per la gestione degli appuntamenti con modalità manuale.

### Funzionalità principali

- Componente **FzAppointments** per la selezione e gestione di slot per appuntamenti (LIB-2360)
- **Modalità manuale** per consentire la selezione libera degli orari (LIB-276)
- Integrazione con FzAlert, FzButton, FzRadio e FzContainer

### Patch Changes

- Updated dependencies
  - @fiscozen/alert@1.0.0
  - @fiscozen/button@1.0.1
  - @fiscozen/radio@1.0.0
  - @fiscozen/container@0.4.1
