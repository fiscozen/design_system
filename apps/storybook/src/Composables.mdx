# Composables

Reusable Vue composables provided by `@fiscozen/composables`. Each composable encapsulates a specific concern and can be used independently.

## Installation

```bash
npm install @fiscozen/composables
```

## Optional Dependencies

Some composables have optional peer dependencies:

- **vue-router** (optional): Required by `useQueryString` when using `provideQueryStringRoute()` for Vue Router integration. Without it, the composable falls back to the browser History API.
