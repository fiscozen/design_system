# FzPdfViewer

PDF viewer component with toolbar controls for zoom and page navigation. Supports two toolbar variants, configurable positioning, drag-to-pan, and an environment prop for frontoffice/backoffice sizing.

## Features

- **Two Toolbar Variants**: `basic` (zoom + page navigation) and `advanced` (adds view mode toggle, download, and reset zoom)
- **Toolbar Positioning**: Toolbar can be placed at the bottom (default) or top; when at top, it is hidden and revealed on hover
- **Environment Sizing**: `frontoffice` (smaller text) and `backoffice` (larger text) variants
- **Drag to Pan**: Click and drag to pan when the PDF overflows its container
- **View Mode Toggle**: Switch between PDF and XML views (advanced toolbar only)
- **Keyboard Accessible**: All controls are keyboard navigable

## Installation

```bash
npm install @fiscozen/pdf-viewer
```

## Basic Usage

```vue
<script setup lang="ts">
import { FzPdfViewer } from '@fiscozen/pdf-viewer'
</script>

<template>
  <FzPdfViewer src="https://example.com/document.pdf" />
</template>
```

## Variants

### Basic Toolbar (Default)

Shows zoom controls and page navigation side by side.

```vue
<FzPdfViewer
  src="https://example.com/document.pdf"
  toolbarVariant="basic"
/>
```

### Advanced Toolbar

Adds a PDF/XML view mode toggle on the left, page navigation and zoom controls in the center, and download + reset zoom buttons on the right.

```vue
<FzPdfViewer
  src="https://example.com/document.pdf"
  toolbarVariant="advanced"
  @download="handleDownload"
/>
```

### Toolbar at Top

When `toolbarPosition="top"`, the toolbar is rendered above the PDF and is only visible on hover. The background transitions from grey (matching the PDF area) to white on hover.

```vue
<FzPdfViewer
  src="https://example.com/document.pdf"
  toolbarPosition="top"
/>
```

### Backoffice Environment

```vue
<FzPdfViewer
  src="https://example.com/document.pdf"
  environment="backoffice"
/>
```

### Custom Dimensions

```vue
<FzPdfViewer
  src="https://example.com/document.pdf"
  height="600px"
  width="800px"
/>
```

### Starting at a Specific Page and Scale

```vue
<FzPdfViewer
  src="https://example.com/document.pdf"
  :initialPage="3"
  :initialScale="1.5"
/>
```

### Tracking View Mode

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { FzPdfViewer } from '@fiscozen/pdf-viewer'

const viewMode = ref<'pdf' | 'xml'>('pdf')
</script>

<template>
  <FzPdfViewer
    src="https://example.com/document.pdf"
    toolbarVariant="advanced"
    v-model:viewMode="viewMode"
  />
  <p>Current view: {{ viewMode }}</p>
</template>
```

## Accessibility

- All toolbar buttons have descriptive `aria-label` attributes (`"Zoom in"`, `"Zoom out"`, `"Next page"`, `"Previous page"`, `"Download"`, `"Reset zoom"`)
- Buttons are disabled at scale/page boundaries and convey their state via the native `disabled` attribute
- All controls are reachable and activatable via keyboard (Tab, Enter, Space)
- The page counter and scale display use `data-testid` attributes for automated testing
