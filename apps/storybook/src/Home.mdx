## Welcome

Welcome to the Fiscozen Design System documentation.

Here, you can find all the information you need to start using our design system. This includes installation instructions, component documentation, and usage examples.

## Overview

Fiscozen Design System is a set of components for Vue.js. The primary goal is to increase developer productivity by offering reusable and simple-to-integrate components.

e.g.

```tsx
<script setup>
    import { FzButton } from '@fiscozen/button';
</script>
<template>
    <FzButton ... />
</template>
```
## Installation

Our design system is modular, providing individual packages for each component. This allows developers to selectively include only the components they need, avoiding the overhead of importing the entire library.

Each package is scoped under **@fiscozen** followed by the specific component name.
