# FzActionlist

> ⚠️ **Deprecated for external use — internal-only until consumers migrate.** New code must use [`FzActionList`](./?path=/docs/navigation-fzactionlist-documentation--docs) (with `FzActionSection` and `FzAction`) from `@fiscozen/action` instead. This package stays published on npm and remains buildable because it is still consumed internally by **`@fiscozen/table`**. It cannot be removed until that consumer migrates. No new features will be developed and no bug fixes will be backported.

A vertical list of interactive actions or options, composed of items presented in a consistent, single-column format.

## Migration to @fiscozen/action

`FzActionlist` (from `@fiscozen/actionlist`, lowercase `l`) maps to a composition of three components from `@fiscozen/action`: `FzActionList` + `FzActionSection` + multiple `FzAction` children.

```vue
<!-- Before: @fiscozen/actionlist -->
<FzActionlist
  :label="label"
  :items="items"
  @fzaction:click="onClick"
/>

<!-- After: @fiscozen/action -->
<FzActionList :listClass="listClass">
  <FzActionSection :label="label">
    <FzAction
      v-for="(item, index) in items"
      :key="index"
      :type="item.type === 'link' ? 'link' : 'action'"
      variant="textLeft"
      :label="item.label"
      v-bind="item"
      @click="onClick(index, item)"
    />
  </FzActionSection>
</FzActionList>
```

    <tr>
