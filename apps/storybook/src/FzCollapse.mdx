# FzCollapse

A collapsible panel component built on native `<details>/<summary>` HTML elements. Supports two visual variants (`section` and `button`), an optional leading icon with content indentation, and subtitle text.

## Features

- Two variants: `section` (larger, with optional subtitle) and `button` (compact, with right content slot)
- Built on semantic `<details>/<summary>` for native accessibility
- Optional leading icon with automatic content indentation
- Subtitle visible only when collapsed (section variant)
- `v-model:open` for controlled open/close state
- Works standalone or inside `FzAccordion` for coordinated panels

## Installation

```bash
npm install @fiscozen/collapse
```

## Basic Usage

```vue
<script setup>
import { FzCollapse } from '@fiscozen/collapse'
import { ref } from 'vue'

const isOpen = ref(false)
</script>

<template>
  <FzCollapse
    title="Section title"
    subtitle="Additional info shown when collapsed"
    icon="face-smile"
    v-model:open="isOpen"
  >
    <template #content>
      <p>Your content here.</p>
    </template>
  </FzCollapse>
</template>
```

## Examples

### Section Variant (default)

The section variant uses larger typography (semibold 17px title) and supports a subtitle that is visible only when the panel is closed.

```vue
<template>
  <FzCollapse
    title="Customer Details"
    subtitle="Mario Rossi - MR di Mario Rossi"
    icon="user"
    v-model:open="isOpen"
  >
    <template #content>
      <div class="grid grid-cols-2 gap-24">
        <div>
          <span class="text-grey-500">Nome e Cognome</span>
          <p>Mario Rossi</p>
        </div>
        <div>
          <span class="text-grey-500">Codice Fiscale</span>
          <p>RSSMRA74A01F205Z</p>
        </div>
      </div>
    </template>
  </FzCollapse>
</template>
```

### Button Variant

The button variant is compact (regular 16px title) and supports a `#rightContent` slot for badges, prices, or other inline information.

```vue
<template>
  <FzCollapse
    title="Order total"
    variant="button"
    icon="cart-shopping"
    v-model:open="isOpen"
  >
    <template #rightContent>
      <span class="text-sm font-semibold">€ 1.250,00</span>
    </template>
    <template #content>
      <p>Order breakdown details here.</p>
    </template>
  </FzCollapse>
</template>
```

### Without Icon

When no `icon` prop is provided, the leading icon is hidden and content is not indented.

```vue
<template>
  <FzCollapse title="Simple collapse" v-model:open="isOpen">
    <template #content>
      <p>No icon, no indentation.</p>
    </template>
  </FzCollapse>
</template>
```

## Accessibility

- Built on native `<details>/<summary>` elements for built-in keyboard support and screen reader announcements
- Summary element is focusable and can be activated with Enter or Space keys
- The `open` attribute on `<details>` communicates expanded/collapsed state to assistive technologies
- Leading icons are decorative and hidden from screen readers via FzIcon's internal `aria-hidden`

## Best Practices

- Use the **section** variant for page sections with rich content (forms, data displays)
- Use the **button** variant for compact inline toggles (prices, summaries, settings)
- Provide a `subtitle` in the section variant to give users a preview of the content when collapsed
- Use the `icon` prop to visually categorize sections; the automatic content indentation aligns the content with the title
- For groups of related collapsible panels, use `FzAccordion` instead of managing open states manually

# FzAccordion

A wrapper component that coordinates multiple `FzCollapse` children. Supports exclusive mode (only one panel open at a time) and multiple mode (any number of panels open simultaneously).

## Installation

`FzAccordion` is exported from the same package:

```bash
npm install @fiscozen/collapse
```

## Basic Usage

```vue
<script setup>
import { FzAccordion, FzCollapse } from '@fiscozen/collapse'
import { ref } from 'vue'

const open1 = ref(false)
const open2 = ref(false)
const open3 = ref(false)
</script>

<template>
  <FzAccordion>
    <FzCollapse title="Section 1" icon="user" v-model:open="open1">
      <template #content><p>Content 1</p></template>
    </FzCollapse>
    <FzCollapse title="Section 2" icon="gear" v-model:open="open2">
      <template #content><p>Content 2</p></template>
    </FzCollapse>
    <FzCollapse title="Section 3" icon="file" v-model:open="open3">
      <template #content><p>Content 3</p></template>
    </FzCollapse>
  </FzAccordion>
</template>
```

## Examples

### Exclusive Mode (default)

Only one panel can be open at a time. Opening a new panel automatically closes the previously open one.

```vue
<template>
  <FzAccordion>
    <FzCollapse title="Panel A" v-model:open="openA">
      <template #content><p>Content A</p></template>
    </FzCollapse>
    <FzCollapse title="Panel B" v-model:open="openB">
      <template #content><p>Content B</p></template>
    </FzCollapse>
  </FzAccordion>
</template>
```

### Multiple Mode

Any number of panels can be open simultaneously.

```vue
<template>
  <FzAccordion :multiple="true">
    <FzCollapse title="Panel A" v-model:open="openA">
      <template #content><p>Content A</p></template>
    </FzCollapse>
    <FzCollapse title="Panel B" v-model:open="openB">
      <template #content><p>Content B</p></template>
    </FzCollapse>
  </FzAccordion>
</template>
```

## Accessibility

- Accordion coordination is implemented via Vue's provide/inject; no additional ARIA attributes are added beyond what each FzCollapse provides natively
- Each panel remains independently focusable and keyboard-operable
