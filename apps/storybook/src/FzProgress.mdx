# FzProgress

Animated loading spinner indicator component. Wraps FzIcon with spinning animation for use in loading states.

## Features

- Animated spinner with smooth rotation
- Customizable icon size
- Custom animation timing
- FontAwesome icon integration
- Accessible by default (FontAwesome handles ARIA)

## Installation

```bash
npm install @fiscozen/progress
```

## Basic Usage

```vue
<script setup>
import { FzProgress } from '@fiscozen/progress'
</script>

<template>
  <FzProgress />
</template>
```

## Accessibility

FzProgress is a polite status indicator:

- The wrapper has `role="status"` (implicit `aria-live="polite"`), so screen readers announce the loading state when the spinner enters the DOM
- `aria-label` defaults to `'Caricamento…'`, overridable via the `label` prop, and is mirrored as visually-hidden text
- The spin animation is disabled under `prefers-reduced-motion: reduce`

The container that toggles the spinner should set `aria-busy="true"` while loading, so the surrounding content is marked as not yet ready.
