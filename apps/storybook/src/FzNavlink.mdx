# FzNavlink / FzRouterNavlink

> ⚠️ **Deprecated for external use — internal-only until consumers migrate.** New code must use [`FzAction`](./?path=/docs/navigation-fzaction-documentation--docs) from `@fiscozen/action` (with `variant="textLeft"`) instead. This package stays published on npm and remains buildable because it is still consumed internally by **`@fiscozen/actionlist`** and **`@fiscozen/navlist`** (both themselves deprecated). It cannot be removed until those consumers migrate. No new features will be developed and no bug fixes will be backported.

This package exports two components:
- `FzNavlink` — a button-like navigation item with optional icon and label.
- `FzRouterNavlink` — the same, wrapped around `vue-router`'s `<router-link>`.

Both map to `FzAction` with `variant="textLeft"`, distinguished by the `type` prop.

## Migration to @fiscozen/action

### Button variant (`FzNavlink`)

```vue
<!-- Before -->
<FzNavlink
  :iconName="iconName"
  :iconVariant="iconVariant"
  :disabled="disabled"
  @click="onClick"
>
  {{ label }}
</FzNavlink>

<!-- After -->
<FzAction
  type="action"
  variant="textLeft"
  :label="label"
  :iconLeftName="iconName"
  :iconLeftVariant="iconVariant"
  :disabled="disabled"
  @click="onClick"
/>
```

### Router-link variant (`FzRouterNavlink`)

```vue
<!-- Before -->
<FzRouterNavlink
  :meta="{ name: 'home' }"
  :iconName="iconName"
  :disabled="disabled"
>
  {{ label }}
</FzRouterNavlink>

<!-- After -->
<FzAction
  type="link"
  variant="textLeft"
  :label="label"
  :iconLeftName="iconName"
  :to="{ name: 'home' }"
  :disabled="disabled"
/>
```
